# Superstore Sales Analysis

Exploratory data analysis on 9,994 retail orders using Python, pandas, and numpy.

## Key Findings
- Heavy discounting (>40%) causes average losses of $134 per order
- Furniture margin is 2.49% vs Technology at 17.40%
- Large orders are 59x more profitable than small orders
- 3D printer product line losing $12,000+ total

## Skills Demonstrated
- pandas: groupby, merge, pivot_table, filtering, data cleaning
- numpy: arrays, percentiles, correlation, np.where
- matplotlib + seaborn: bar charts, scatter plots, heatmaps, distributions
- Data storytelling: translating numbers into business recommendations

## Tools
Python 3.11 · pandas · numpy · matplotlib · seaborn · Jupyter Notebook
